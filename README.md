# OCMethodTrace
OCMethodTrace - Trace any Objective-C method calls
跟踪 Objective-C实例(类)中任何方法

================================================================================
功能
================================================================================
1. 支持任意OC实例(类)方法的跟踪打印
2. 避免跟踪基类super调用导致的递归死循环
3. 避免输出打印时调用-[description]导致的递归死循环
